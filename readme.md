Analiza patentowa
=================

Uruchamianie skryptów

```sh
python main.py [plik].[funkcja]
```

Generowanie pliku z wynikami analizy patentowej:

```sh
python docs.py
```

Generowanie podglądu analizy z pojedynczego skryptu

```sh
python docs.py [plik]
```
